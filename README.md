# Vaanavil Tamil Reader - Mobile

This is the mobile version of the Vaanavil Tamil Reader application, built using Capacitor.

## Automated Builds
This repository uses GitHub Actions to build an Android APK on every push.

### How to get the APK:
1. Go to the **Actions** tab.
2. Select the latest successful workflow run.
3. Download the `vaanavil-reader-debug-apk` from the Artifacts section.
